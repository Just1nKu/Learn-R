# Biomedical Applications of R

R is widely used in clinical research, genomics, and epidemiology. This module introduces survival analysis, clinical data workflows, and the Bioconductor ecosystem.

## Survival Analysis

Survival analysis models time-to-event data (death, relapse, hospital discharge).

### Installation

```r
install.packages("survival")
install.packages("survminer")
library(survival)
library(survminer)
```

### The Surv Object

```r
data(lung)  # NCCTG lung cancer data

# Create survival object: time + event indicator (1 = event)
surv_obj <- Surv(time = lung$time, event = lung$status)
```

### Kaplan-Meier Estimator

```r
km_fit <- survfit(Surv(time, status) ~ sex, data = lung)
summary(km_fit)

# Plot with survminer
ggsurvplot(
  km_fit,
  data      = lung,
  pval      = TRUE,
  conf.int  = TRUE,
  risk.table = TRUE,
  legend.labs = c("Male", "Female"),
  title     = "Survival by Sex — NCCTG Lung Cancer"
)
```

### Log-Rank Test

```r
survdiff(Surv(time, status) ~ sex, data = lung)
# p < 0.05 indicates groups differ significantly
```

### Cox Proportional Hazards Model

```r
cox_model <- coxph(
  Surv(time, status) ~ age + sex + ph.ecog,
  data = lung
)
summary(cox_model)

# Hazard ratios with 95% CI
exp(coef(cox_model))
exp(confint(cox_model))
```

Check proportional hazards assumption:

```r
cox.zph(cox_model)
```

## Bioconductor Basics

Bioconductor provides packages for genomic data analysis.

### Installation


### RNA-seq Differential Expression with DESeq2

```r
library(DESeq2)

# Create DESeqDataSet from count matrix
dds <- DESeqDataSetFromMatrix(
  countData = count_matrix,
  colData   = sample_metadata,
  design    = ~ condition
)

# Filter low-count genes
dds <- dds[rowSums(counts(dds)) >= 10, ]

# Run analysis
dds <- DESeq(dds)
res <- results(dds, contrast = c("condition", "treated", "control"))
```

### Volcano Plot

```r
library(EnhancedVolcano)

EnhancedVolcano(
  res,
  lab     = rownames(res),
  x       = "log2FoldChange",
  y       = "pvalue",
  title   = "Treated vs Control",
  pCutoff = 0.05,
  FCcutoff = 1.5
)
```
